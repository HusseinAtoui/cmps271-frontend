General Submission Guidelines

Afterthoughts is an english published in print annually. The call for submissions for our print edition is announced on social media channels and on this website. All submissions for this call will be considered for print and online publication. To submit, see our Submittable portal for open calls.

If you have questions, please send an e-mail to info@rustedradishes.com. Any work submitted not in line with these guidelines will not be accepted, nor read.

Submission Requirements

Please follow these guidelines when submitting:

  • Attachment: Each writing piece should be attached in a separate document in .doc form. Each art piece should be sent as a low-res attachment.
  • Bio: Please provide a brief bio written in 3rd person that does not exceed 80 words. Indicate whether you are a student, faculty, or alum of AUB; and/or your connection to Lebanon or the region. We reserve the right to edit your bio for syntax and grammatical correctness.
  • Formatting: Use a 12-pt font and double-space each prose piece. Please retain the original aesthetic in your poetry.
  • Multiple Submissions: Check the guidelines specific to each genre for the maximum allowance.
  • File Naming: Name each submission in the format ‘title_genre’. Include your full name and e-mail address at the top of the first page of the actual submission file.
  • Simultaneous Submissions: These are welcome. If you need to withdraw work, please write to submit@rustedradishes.com or use the Submittable submissions manager. For poetry, leave a note specifying which poem you’d like to withdraw.
